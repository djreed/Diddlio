# HTML5_Gaming

Agar.io clone written with Socket.IO and Phaser

Try online: http://mickaelvillers.me:3000/index.html

## Installation
#### Downloading the Dependencies
After cloning the source, you need to run the following command.

```
npm install
bower install
```

#### Running the Server
```
node server.js
```

#### Playing
`http://localhost:3000`.
